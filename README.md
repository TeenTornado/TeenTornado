### 🔐 VibeCheck — AI-Powered Vulnerability Scanner
> *Minor Project, SRMIST · 2024–25*

Not a toy. A real hybrid engine.

- **47 regex patterns** + **128 Semgrep OWASP rules** running in parallel
- **Random Forest Regressor** CVSS scoring model trained on **180,000+ CVE records**
- SHA-256 deduplication to kill false positive noise
- Google Gemini 2.0 Pro as the AI remediation layer
- **Precision: 0.978 · Recall: 0.880 · F1: 0.926** on 100 custom test cases
- Stack: Next.js 14, Monaco Editor, Python backend

> Built because "AI finds bugs" tools either hallucinate everything or catch nothing. This one doesn't.
